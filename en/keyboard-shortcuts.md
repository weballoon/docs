# Keyboard shortcuts

Shortcuts below match the **current** weballoon implementation. They assume you are **not** typing in a text field, search box, or other editable control—when you are, letter shortcuts may go to the site instead.

## Workspace switching

| Action | Shortcut |
|--------|-----------|
| **Next workspace** (cycle) | **Shift+W** |
| **Jump to workspace 1–9** | **Ctrl+1** … **Ctrl+9** on Windows/Linux, **⌘1** … **⌘9** on macOS — **only while focus is inside an open web app** (so the key does not conflict with weballoon’s own screens). |

While **Shift** is held after cycling with **Shift+W**, a small **workspace switcher** overlay can appear until you release **Shift**.

From **My Apps**, **Catalog**, or **Settings**, use the **workspace** button or **Shift+W** to change context if number shortcuts do not respond.

## Dock

- **Workspace** button tooltip documents **Shift+W** for workspaces.
- **Notifications** bell tooltip documents **Shift+N**.

## Title bar

Navigation buttons (**Back**, **Home**, **Forward**, **Reload**) are **mouse / pointer** only in the current UI—there is no documented separate keyboard row for them.

If a shortcut does nothing, click the weballoon background or title bar first so focus is not inside a site’s input field.
