# Task Manager

Open **Settings** and select the **Task Manager** tab.

## What it shows

- **Totals** for weballoon: **CPU**, **RAM**, **storage**, process counts, and how many apps are running.
- A **sortable table** of your applications with **CPU**, **RAM**, **storage**, **workspace** name, and whether each app is **Open** or **Idle** (based on whether an active web session is registered for that app).

Data **refreshes automatically every few seconds**; you can also use **Refresh** manually.

## Using it well

- Sort by **memory** or **CPU** to find heavy tabs.
- If numbers look stuck after a big change, click **Refresh**.

If you have no apps yet, the panel asks you to add apps first.
