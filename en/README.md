# weballoon — User guide

weballoon is a desktop workspace for web apps: separate contexts, your data on your machine, and optional cloud sync for your setup—not your browsing logins.

This guide is for **everyday use**. It matches the app as shipped; if something changes in a future version, prefer what you see on screen.

**Languages:** this folder (`en`) is English. Translated copies of the same files live in `es/`, `pt/`, `fr/`, `de/`, `ru/`, and `it/` (same filenames for easy cross-linking).

## Quick facts

- **Platforms:** macOS, Windows, and Linux (see [Installation](installation.md)).
- **Free tier:** Up to **10 apps** and **2 workspaces** (excluding the default view).
- **Sync:** Optional; syncs app and workspace **configuration**, not site passwords or browsing contents (see [Account & sync](account-and-cloud-sync.md)).

If anything here disagrees with the app, trust the app.
