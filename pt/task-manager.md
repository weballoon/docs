# Gestor de tarefas

Abra **Definições** e selecione o separador **Gestor de tarefas**.

## O que mostra

- **Totais** do weballoon: **CPU**, **RAM**, **armazenamento**, contagens de processos e quantas apps estão a correr.
- Uma **tabela ordenável** das suas aplicações com **CPU**, **RAM**, **armazenamento**, nome do **espaço de trabalho** e se cada app está **Aberta** ou **Inativa** (consoante haja uma sessão web ativa registada para essa app).

Os dados **atualizam-se automaticamente a cada poucos segundos**; também pode usar **Atualizar** manualmente.

## Bom uso

- Ordene por **memória** ou **CPU** para encontrar separadores pesados.
- Se os números parecerem bloqueados após uma grande mudança, clique em **Atualizar**.

Se ainda não tiver apps, o painel pede que adicione apps primeiro.
