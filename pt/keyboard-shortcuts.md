# Atalhos de teclado

Os atalhos abaixo correspondem à implementação **atual** do weballoon. Pressupõem que **não** está a escrever num campo de texto, caixa de pesquisa ou outro controlo editável — nesse caso, as letras podem ir para o site.

## Mudança de espaço de trabalho

| Ação | Atalho |
|--------|--------|
| **Próximo espaço de trabalho** (ciclo) | **Shift+W** |
| **Ir para o espaço de trabalho 1–9** | **Ctrl+1** … **Ctrl+9** no Windows/Linux, **⌘1** … **⌘9** no macOS — **apenas com o foco dentro de uma app web aberta** (para não conflituar com os ecrãs próprios do weballoon). |

Depois de percorrer com **Shift+W**, enquanto mantém **Shift** pode aparecer uma pequena **sobreposição** de mudança de espaço até soltar **Shift**.

Em **As minhas apps**, **Catálogo** ou **Definições**, use o botão **espaço de trabalho** ou **Shift+W** se os atalhos com números não responderem.

## Doca

- A dica do botão **Espaço de trabalho** documenta **Shift+W** para espaços.
- A dica da campainha **Notificações** documenta **Shift+N**.

## Barra de título

Os botões de navegação (**Recuar**, **Início**, **Avançar**, **Recarregar**) são **apenas rato / ponteiro** na interface atual — não há linha de teclado documentada à parte.

Se um atalho não fizer nada, clique primeiro no fundo do weballoon ou na barra de título para o foco não estar num campo do site.
