# weballoon — Guia do utilizador

O weballoon é um ambiente de trabalho no ambiente de trabalho para aplicações web: contextos separados, os seus dados na sua máquina e sincronização opcional na nuvem para a sua configuração — não para os seus logins de navegação.

Este guia é para o **uso quotidiano**. Corresponde à aplicação publicada; se algo mudar numa versão futura, confie no que vê no ecrã.

**Idiomas:** esta pasta (`pt`) contém a documentação em **português**. A versão em inglês está em `en/`.

## Factos rápidos

- **Plataformas:** macOS, Windows e Linux (ver [Instalação](installation.md)).
- **Plano gratuito:** até **10 apps** e **2 espaços de trabalho** (excluindo a vista predefinida).
- **Sincronização:** opcional; sincroniza a **configuração** de apps e espaços de trabalho, não palavras-passe de sites nem conteúdo de navegação (ver [Conta e sincronização](account-and-cloud-sync.md)).

Se algo aqui não coincidir com a app, confie na app.
