# weballoon — Guida utente

weballoon è uno spazio di lavoro desktop per app web: contesti separati, i tuoi dati sul tuo computer e sincronizzazione cloud opzionale per la **configurazione**, non per i login di navigazione.

Questa guida è per l’**uso quotidiano**. Corrisponde all’app pubblicata; se qualcosa cambia in una versione futura, fai affidamento a ciò che vedi sullo schermo.

**Lingue:** questa cartella (`it`) contiene la documentazione in **italiano**. La versione inglese è in `en/`.

## In sintesi

- **Piattaforme:** macOS, Windows e Linux (vedi [Installazione](installation.md)).
- **Piano gratuito:** fino a **10 app** e **2 spazi di lavoro** (esclusa la vista predefinita).
- **Sincronizzazione:** opzionale; sincronizza la **configurazione** di app e spazi, non password dei siti né contenuti di navigazione (vedi [Account e sincronizzazione](account-and-cloud-sync.md)).

Se qualcosa qui non coincide con l’app, ha ragione l’app.
