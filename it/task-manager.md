# Task Manager

Apri **Impostazioni** e seleziona la scheda **Task Manager**.

## Cosa mostra

- **Totali** per weballoon: **CPU**, **RAM**, **archiviazione**, conteggio processi e quante app sono in esecuzione.
- Una **tabella ordinabile** delle tue applicazioni con **CPU**, **RAM**, **archiviazione**, nome dello **spazio di lavoro** e se ogni app è **Aperta** o **Inattiva** (in base alla sessione web registrata).

I dati si **aggiornano automaticamente ogni pochi secondi**; puoi anche usare **Aggiorna** manualmente.

## Buon uso

- Ordina per **memoria** o **CPU** per trovare schede pesanti.
- Se i numeri sembrano bloccati dopo un grande cambiamento, fai clic su **Aggiorna**.

Senza app, il pannello chiede prima di aggiungere le app.
