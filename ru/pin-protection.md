# Защита PIN-кодом

Откройте **Настройки → Защита**.

## Назначение

Можно защитить приложения **6-значным цифровым PIN**.

- Включите **Включить защиту**. Если PIN ещё нет — мастер **настройки**; если есть — нужно **подтвердить** перед включением/выключением.
- **Настроить PIN** или **Сменить PIN** — управление кодом (с подтверждением).

## Блокировка по приложениям

Ниже, в **Защищённые приложения**, включите переключатель для каждого приложения, которое должно запрашивать PIN **при открытии**.

Переключатели по приложениям **неактивны**, пока не включена общая защита (секция визуально приглушена).

## Когда запрашивается PIN

При открытии **защищённого** приложения показывается экран **аутентификации**, пока PIN не введён верно. После одного разблокирования приложение остаётся разблокированным **до конца сессии** weballoon (не нужно вводить PIN при каждом переключении туда-обратно в одном запуске).

## Забыли PIN

В интерфейсе нет «мастер-ключа». Нужен официальный путь восстановления (поддержка или документированные шаги). **Не** подбирайте PIN наугад; храните резервную копию кода в безопасном месте.
