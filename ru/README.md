# weballoon — Руководство пользователя

weballoon — это рабочее пространство на рабочем столе для веб-приложений: отдельные контексты, ваши данные на вашем компьютере и при желании облачная синхронизация **настроек**, а не паролей от сайтов и содержимого браузера.

Это руководство для **повседневного использования**. Оно соответствует опубликованному приложению; если в будущей версии что-то изменится, ориентируйтесь на то, что видите на экране.

**Языки:** в этой папке (`ru`) — документация на **русском**. Английская версия находится в `en/`.

## Кратко

- **Платформы:** macOS, Windows и Linux (см. [Установка](installation.md)).
- **Бесплатный тариф:** до **10 приложений** и **2 рабочих пространства** (не считая вида по умолчанию).
- **Синхронизация:** по желанию; синхронизируется **конфигурация** приложений и пространств, не пароли от сайтов и не содержимое браузера (см. [Аккаунт и синхронизация](account-and-cloud-sync.md)).
