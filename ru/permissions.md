# Разрешения приложений (камера, микрофон, экран, геолокация)

Откройте **Настройки → Конфиденциальность и данные**, прокрутите до **Чувствительный доступ** (название может слегка отличаться; перечислены камера, микрофон, демонстрация экрана и геолокация).

## По умолчанию

**Камера**, **микрофон**, **демонстрация экрана** и **геолокация** **заблокированы**, пока не разрешите **для каждого приложения**. Изменения применяются к открытым окнам приложений сразу.

## Переключатели по приложению

Для каждой установленной приложения — компактные переключатели:

- **Камера**
- **Микрофон**
- **Экран**
- **Местоположение**

Включайте только для доверенных приложений. Видеозвонки, голосовые заметки, карты и захват экрана требуют соответствующих переключателей.

Без приложений раздел просит сначала добавить приложения.

## macOS

На macOS заданы тексты использования для камеры, микрофона и геолокации — система покажет диалоги, когда сайту это нужно **и** ваш переключатель разрешает.
