# Устранение неполадок

Спокойные шаги. Если не помогло — укажите **версию weballoon** (Настройки → Обновления приложения) и **ОС** при обращении в поддержку.

## Magic link не привязывает

- Ссылка открыта на **том же компьютере**, где установлен weballoon.
- macOS / Windows: ссылки **`weballoon://`** открывают weballoon (обработчик по умолчанию).
- **Отправить ссылку снова** в **Настройках → Аккаунт** после паузы.
- Папка «Спам».

## «Достигнут лимит бесплатного тарифа»

**10 приложений** или попытка создать больше **2** пространств на Free. Удалите приложение/пространство или перейдите на платный тариф в **Тарифах** ([Тарифы и оплата](plans-and-billing.md)).

## Сайт сломан или пустой

- **Обновить** в заголовке.
- **Настройки → Блокировщик рекламы** — отключить **только для этого приложения**.
- **Настройки → Прокси** — проверить хост, порт, исключения; сохранить и дать перезагрузиться.

## Нет уведомлений

- **Работа в фоне** для приложения в **Настройках → Уведомления**.
- Сайт должен использовать **веб-уведомления**.
- Если вы **внутри** этого приложения с фокусом, дубли во внутреннем списке могут не создаваться.

## Сочетание не работает

- Клик вне полей сайта (заголовок или chrome weballoon), снова попробовать ([Сочетания клавиш](keyboard-shortcuts.md)).
- **Ctrl/Cmd+цифра** — только **внутри** открытого веб-приложения; с **Мои приложения** / **Настройки** — кнопка пространства или **Shift+W**.

## Высокая загрузка CPU или RAM

- **Настройки → Диспетчер задач** — сортировка.
- Меньше **Работа в фоне**.
- Закрывайте ненужные виды приложений.

## Обновления не появляются

Карточка **Обновления** объясняет, когда автообновления **отключены** (dev-сборки, часть Linux, переменные среды). По возможности установите **релиз**.

## Окно закрыто, weballoon «ещё работает» (Windows / Linux)

**Иконка в трее**. **Правый клик → Выход** для полного выхода или клик для окна ([Обзор настроек](settings.md)).

## Тема не совпала с системой

Тема weballoon **ручная** (тёмная/светлая), не «как система» в текущем селекторе — выберите в **Внешний вид**.
