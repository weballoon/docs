# Частые вопросы

## weballoon заменяет обычный браузер?

Дополняет. Удобен для **стабильных наборов** веб-приложений как инструментов рабочего стола с **изоляцией** и **пространствами**.

## Рабочие пространства делят cookies?

**Нет.** Изоляция **по приложению**. Пространство — только **организация**.

## Что уходит в облако?

**Конфигурация приложений и рабочих пространств** при включённой синхронизации — не пароли от сайтов и не содержимое браузера (по тексту в приложении). См. [Аккаунт и облачная синхронизация](account-and-cloud-sync.md).

## Два аккаунта Gmail / Slack / и т. д.?

Да, через **два отдельных приложения** (разные имена, URL по правилам сервиса). **Один URL дважды в одном пространстве** нельзя — два пространства или разные URL входа.

## Есть мобильное приложение?

Документация только для **десктопа**.

## Работает ли offline?

Только там, где **сам сайт** поддерживает offline. weballoon не делает произвольные сайты offline-first.

## Где мои данные?

**Настройки → Конфиденциальность и данные** — путь и **Открыть папку**. Данные **в первую очередь локальные**; облако — опционально для конфигурации и оплаты.

## Сколько стоит Pro?

Откройте **Тарифы** в приложении — цены в реальном времени.

## Есть тариф «Team»?

В переводах есть **метки** для командного биллинга, но встроенный список в коде — **Free**, **Pro** и **Pro (год)**. Смотрите, что реально показывает **Тарифы** для вашего аккаунта.
