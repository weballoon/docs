# Диспетчер задач

Откройте **Настройки** и вкладку **Диспетчер задач**.

## Что показывается

- **Итоги** по weballoon: **CPU**, **RAM**, **диск**, число процессов и сколько приложений запущено.
- **Сортируемая таблица** приложений с **CPU**, **RAM**, **диском**, именем **рабочего пространства** и статусом **Открыто** или **Простой** (есть ли активная веб-сессия).

Данные **обновляются каждые несколько секунд**; есть кнопка **Обновить**.

## Советы

- Сортируйте по **памяти** или **CPU**, чтобы найти тяжёлые вкладки.
- После больших изменений нажмите **Обновить**, если цифры «застыли».

Без приложений панель просит сначала добавить приложения.
