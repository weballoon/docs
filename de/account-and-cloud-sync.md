# Kontoverknüpfung und Cloud-Sync

## Kontoverknüpfung (Magic Link)

Die Verknüpfung bindet dieses **Gerät** an eine **E-Mail** beim weballoon-Dienst, damit Abrechnung und optionaler Sync funktionieren.

1. **Einstellungen → Konto** öffnen (oder Onboarding abschließen).
2. **E-Mail-Adresse** und optional **Gerätebezeichnung** eingeben (z. B. `desktop-mac`).
3. **Magic Link senden** klicken.
4. E-Mail prüfen und **Link anklicken**. weballoon schließt die Verknüpfung automatisch ab, wenn der Link die App öffnet (über `weballoon://`).

Während des Wartens können Sie **Link erneut senden** (mit kurzer Wartezeit). Verbunden können Sie **Gerät trennen**.

Ohne Verknüpfung können Sie im Onboarding **nur lokal** nutzen und später in Einstellungen verknüpfen.

## Cloud-Sync

Unter **Einstellungen → Konto** erklärt der Bereich **Cloud-Sync**:

- Sync speichert Ihre **Apps und Arbeitsbereiche** in der Cloud **ohne** Browser-Daten oder Website-Logins (In-App-Text: keine Browserdaten oder Logins im Snapshot).
- **Cloud-Sync** lässt sich ein- und ausschalten.
- **Sync beim Beenden** führt beim Schließen von weballoon einen weiteren Sync der Gerätedaten aus (wenn aktiviert).

**Aus Cloud abrufen** und **Zu Cloud senden** dienen der manuellen Aktualisierung. Das Panel zeigt Namespace, Version und Zeitstempel des letzten Push/Pull, sofern vorhanden.

Sync erfordert ein **verbundenes** Konto. Fehler erscheinen im selben Panel.
