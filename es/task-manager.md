# Administrador de tareas

Abre **Ajustes** y selecciona la pestaña **Administrador de tareas**.

## Qué muestra

- **Totales** de weballoon: **CPU**, **RAM**, **almacenamiento**, número de procesos y cuántas apps están en ejecución.
- Una **tabla ordenable** de tus aplicaciones con **CPU**, **RAM**, **almacenamiento**, nombre del **espacio de trabajo** y si cada app está **Abierta** o **Inactiva** (según si hay una sesión web activa registrada para esa app).

Los datos **se actualizan solos cada pocos segundos**; también puedes usar **Actualizar** manualmente.

## Buen uso

- Ordena por **memoria** o **CPU** para encontrar pestañas pesadas.
- Si los números parecen congelados tras un cambio grande, pulsa **Actualizar**.

Si aún no tienes apps, el panel te pide que añadas apps primero.
