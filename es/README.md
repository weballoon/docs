# weballoon — Guía de usuario

weballoon es un espacio de trabajo en el escritorio para aplicaciones web: contextos separados, tus datos en tu equipo y sincronización en la nube opcional para tu configuración, no para tus inicios de sesión de navegación.

Esta guía es para el **uso cotidiano**. Coincide con la aplicación publicada; si algo cambia en una versión futura, confía en lo que ves en pantalla.

**Idiomas:** esta carpeta (`es`) es la documentación en **español**. La versión en inglés está en `en/`.

## Datos rápidos

- **Plataformas:** macOS, Windows y Linux (ver [Instalación](installation.md)).
- **Plan gratuito:** hasta **10 apps** y **2 espacios de trabajo** (sin contar la vista predeterminada).
- **Sincronización:** opcional; sincroniza la **configuración** de apps y espacios de trabajo, no contraseñas de sitios ni contenido de navegación (ver [Cuenta y sincronización](account-and-cloud-sync.md)).

Si algo aquí no coincide con la app, confía en la app.
