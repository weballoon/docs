# weballoon — Guide utilisateur

weballoon est un espace de travail sur le bureau pour les applications web : des contextes séparés, vos données sur votre machine, et une synchronisation cloud optionnelle pour votre configuration — pas pour vos identifiants de navigation.

Ce guide s’adresse à l’**usage quotidien**. Il correspond à l’application telle que publiée ; si quelque chose change dans une version ultérieure, privilégiez ce que vous voyez à l’écran.

**Langues :** ce dossier (`fr`) contient la documentation en **français**. La version anglaise se trouve dans `en/`.

## En bref

- **Plateformes :** macOS, Windows et Linux (voir [Installation](installation.md)).
- **Offre gratuite :** jusqu’à **10 apps** et **2 espaces de travail** (hors vue par défaut).
- **Synchronisation :** optionnelle ; synchronise la **configuration** des apps et espaces, pas les mots de passe des sites ni le contenu de navigation (voir [Compte et synchronisation](account-and-cloud-sync.md)).

En cas de divergence avec l’application, faites confiance à l’application.
