# Bloqueur de publicités

Ouvrez **Réglages → Bloqueur de publicités**.

## Interrupteur global

**Activer le bloqueur global de publicités** active ou désactive le blocage réseau pour toutes les apps qui respectent le réglage.

Lorsque le blocage est désactivé, une **bannière d’avertissement** indique que les traceurs et publicités redeviennent visibles.

## Exception par app

Sous **Contrôles spécifiques par app**, vous pouvez **désactiver** le blocage pour certaines apps si un site se casse ou se charge mal avec le bloqueur activé.

Les lignes par app sont regroupées par espace de travail comme dans les autres panneaux de réglages.

## Après modification

L’app indique que les changements du bloqueur peuvent nécessiter un **redémarrage ou un rafraîchissement de l’app** pour s’appliquer pleinement aux fenêtres **déjà ouvertes**.

## Ce que ce n’est pas

weballoon ne promet pas de bloquer tous les formats publicitaires ; les sites lourds peuvent encore afficher du contenu « first-party » ressemblant à de la promo. L’objectif indiqué dans l’app est moins de pubs et traceurs, de meilleures performances et une meilleure confidentialité **lorsque** le bloqueur est activé.
