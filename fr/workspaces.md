# Espaces de travail

Les espaces de travail vous aident à **séparer des groupes d’apps** — par exemple outils pro vs perso, ou client A vs client B — sans tout mélanger dans une seule longue liste.

## Vue par défaut vs espaces nommés

- **Vue par défaut** — Apps **non** assignées à un espace nommé. Ce n’est pas un objet que vous supprimez ; il est toujours là.
- **Espaces nommés** — Vous les créez avec un nom. Chacun a sa propre liste d’apps dans le dock et sur **Mes apps** lorsque cet espace est actif.

Seules les apps de l’espace **actif** apparaissent dans le dock et la grille (plus les icônes de navigation habituelles).

## Créer un espace de travail

Cliquez sur le bouton **espace de travail** du dock (affiche la première lettre du nom de l’espace actuel, ou **D** pour Défaut). Dans la fenêtre modale, saisissez un nom et utilisez **Créer**. Les nouveaux espaces apparaissent dans la liste ; cliquez sur l’un pour changer et fermer la modale. Appuyez sur **Échap** pour fermer sans choisir une ligne.

## Supprimer un espace de travail

Dans la modale des espaces, utilisez le contrôle **×** d’une ligne. weballoon demande confirmation. Les apps qui n’étaient que dans cet espace redeviennent **non assignées** (elles apparaissent sous **Vue par défaut**).

## Changer d’espace de travail

- Ouvrez la modale **espace de travail** depuis le dock et choisissez **Vue par défaut** ou un espace.
- Utilisez les **raccourcis clavier** (voir [Raccourcis clavier](keyboard-shortcuts.md)) : par exemple **Maj+W** fait défiler les espaces et affiche un petit sélecteur tant que **Maj** est maintenue.

## Limites (offre Free)

Vous pouvez avoir jusqu’à **2 espaces nommés** en Free. En créer un autre affiche une erreur de limite d’offre ; **Pro** supprime ce plafond (voir [Offres et facturation](plans-and-billing.md)).

## Conseils

- Créez les espaces **avant** d’ajouter des apps si vous voulez que chaque nouvelle installation arrive au bon endroit — les ajouts utilisent l’espace actif.
- La même **URL** ne peut pas exister deux fois dans un espace ; choisissez un autre espace ou supprimez le doublon d’abord.
