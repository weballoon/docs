# Gestionnaire des tâches

Ouvrez **Réglages** et sélectionnez l’onglet **Gestionnaire des tâches**.

## Contenu affiché

- **Totaux** pour weballoon : **CPU**, **RAM**, **stockage**, nombre de processus et nombre d’apps en cours d’exécution.
- Un **tableau triable** de vos applications avec **CPU**, **RAM**, **stockage**, nom d’**espace de travail** et si chaque app est **Ouverte** ou **Inactive** (selon qu’une session web active est enregistrée pour cette app).

Les données **se mettent à jour automatiquement toutes les quelques secondes** ; vous pouvez aussi utiliser **Actualiser** manuellement.

## Bon usage

- Triez par **mémoire** ou **CPU** pour repérer les onglets lourds.
- Si les chiffres semblent bloqués après un gros changement, cliquez sur **Actualiser**.

Si vous n’avez pas encore d’apps, le panneau vous demande d’abord d’ajouter des apps.
